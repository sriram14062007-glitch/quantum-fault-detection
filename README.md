# ⚡ Quantum Fault Detection

## Overview
Hybrid quantum-classical neural network for power system fault detection.

## Features
- PennyLane quantum circuits
- PyTorch hybrid model
- Real-world dataset
- ~98% accuracy

## Structure
- notebooks/ → experiments
- src/ → core code
- results/ → outputs

## Run
pip install -r requirements.txt
